### Maintainability Index

*Oman and Hagemeister, Idaho University 1991*

```
MI = MAX(0, (171 - 5.2 * ln(V) - 0.23 * (G) - 16.2 * ln(LOC))*100 / 171)

// V = Halstead Volume
// G = Cyclomatic Complexity
// LOC = count of source Lines Of Code (SLOC)
```
